# Churn-Prediction-System
This is simple streamlit web integrated app which able to detect whether the employer or customer is likely to churn or not. 

## Key features 
- model built with ANN using tensorflow framework with accuracy of 80%,
- developed web based using streamlit framework and deployed in streamlit.io.could
